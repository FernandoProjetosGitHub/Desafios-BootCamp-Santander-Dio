"h1" é uma tag usada para os títulos principais de uma página.

"h2" é uma tag usada para os títulos secundarios de uma página, seções dentro do h1

"h3" é uma tag usada para subtítulo dos h2 de uma página, sub-seções do h2

"h4" é uma tag usada para subtítulo dos h3 de uma página, uso mais técnico ou específico.

"h5" é uma tag usada para subtítulo dos h4 de uma página, raramente usado.
"h6" é uma tag usada para os títulos de menor nível de uma página, raramente usado.

"p" é uma tag que encapsula um parágrafo

"mark" é uma tag usada para dar destaque em um texto, uma sequência de texto marcado como referência, dando relevância em um contexto particular.
"small" é uma tag usada para representar comentários laterais ou letras pequenas de um texto, usada para dar um sútil destaque informativo a certas palavras.
"i" é uma tag usada para transformar o texto em "itálico"
"u" é uma tag usada comunmente para apontar erros ortográficos, era usada posteriormente como underline, porém foi adicionado essa funcionalidade ao css
"strong" é uma tag usada para aplicar "negrito" em textos para destacar trechos
  • "ul" é uma tag usada para criar listas não ordenadas, geralmente usadas para criar menus em páginas web
    1. "ol" é uma tag usada para criar listas ordenadas, geralmente usadas quando queremos numerar ou apontar quantidades de informações
    2. "a" é uma tag usada para criar links para áreas especícifas da sua página web ou de outras páginas

"hr" é uma tag usada para separar por uma linha horizontal contéudos de textos no corpo da página web

"sub" é uma tag usada para abaixar o nível tipográfico de um elemento em uma página web

"sup" é uma tag usada para aumentar o nível tipográfico de um elemento em uma página web

"blockquote" é uma tag usada para indicar que o texto o qual foi encapsulado é uma longa citação

é processado visualmente pelo recuo

"font" é uma tag, que foi descontinuada com a adição de sua funcionalidade para o css, manipular a font de textos de páginas web

"del" é uma tag usada para representar uma parte do texto que foi exclúida de um documento.

"p" é uma tag usada para indicar um parágrafo em textos.

"abbr" é uma tag usada para representar uma abreviação ou acrônimo ABBR
